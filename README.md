### descrição

O projeto consiste na composição de uma tabela que conta com capitais brasileiras e a distância entre essas capitais. As capitais constantes nesse projeto são:

* Florianópolis
* Fortaleza
* Goiânia
* João Pessoa
* Macapá
* Maceió
* Manaus
